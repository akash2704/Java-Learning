## 🛠️ Installation

### On Ubuntu/Debian:
```bash
sudo apt update
sudo apt install default-jdk -y
java -version
```

### On Windows:
1. Download [Java JDK](https://www.oracle.com/java/technologies/javase-jdk17-downloads.html)  
2. Install it and add it to your **PATH**.  
3. Verify with:
   ```cmd
   java -version
   ```
